Listas ordenadas

são todas aquelas listas onde a ordem dos itens é algo muito importante.

  1. acordar
  2. escovar os dentes
  3. tomar café
  4. sair

dentro da tag ol você pode personalizar ela escolhendo o 'type' e o 'start', ex: type a comecando do 3

  1. acordar
  2. escovar os dentes
  3. tomar café
  4. sair

Listas não ordenadas

chamadas de listas com marcadores, que são aquelas onde a ordem dos itens não influenciará no significado da lista.

  • acordar
  • escovar os dentes
  • tomar café
  • sair

dentro da tag ul você pode personalizar ela escolhendo o 'type = disc, circle ou square'

  • acordar
  • escovar os dentes

Lista de definições

É como se fosse um dicionário, temos os termos e as suas descrições.Toda lista de definições está dentro de uma tag 'dl' '/dl' (definition list) Cada termo é um 'dt' (definition term) e cada descrição é um 'dd' (definition description).

'dt' HTML '/dt'
'dd' Linguagem de marcação para criação do conteúdo de um site '/dd'
